# Dataset setup

Download a season-level match CSV from [Football-Data.co.uk](https://www.football-data.co.uk/englandm.php)
and save it as `data/soccer_matches.csv`.

Required fields: `FTHG`, `FTAG`, `HS`, `AS`, `HST`, `AST`, `HC`, `AC`,
`HY`, `AY`, `HR` and `AR`. Team names and the match date are retained for
interpreting anomalies when available.
